\## How to Access These Datasets



All datasets are publicly available from:



\- \*\*Statistics Canada\*\*: \[https://www.statcan.gc.ca](https://www.statcan.gc.ca)

\- \*\*CMHC\*\*: \[https://www.cmhc.ca](https://www.cmhc.ca)





\## How to Download



1\. Go to \[Statistics Canada](https://www150.statcan.gc.ca/t1/tbl1/en/tv.action?pid=3410013901)

2\. Search for the table numbers listed above

3\. Select "Download selected data (for database loading)"

4\. Save files to `data/raw/` folder
